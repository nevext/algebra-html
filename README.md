## 📋 Sobre o Projeto

Este projeto é uma aplicação web interativa que apresenta a solução do Caso 1 da disciplina de Álgebra Linear do CIESA. A interface permite navegar pelas diferentes etapas da resolução do problema do Índice de Nobreza Ponderado utilizando conceitos de multiplicação de matrizes.

### 🎯 Objetivos

- Apresentar a introdução do caso de forma interativa
- Demonstrar o cálculo à mão e via FigJam
- Exibir o código final em Python com interpretação via Pyodide
- Apresentar a conclusão e análise dos resultados

## 🚀 Tecnologias Utilizadas

- **HTML5** - Estruturação do conteúdo
- **CSS3** - Estilização com design responsivo e animações
- **JavaScript** - Interatividade e gerenciamento de modais
- **Pyodide** - Execução de código Python no navegador
- **Lucide Icons** - Biblioteca de ícones
- **Prism.js** - Syntax highlighting para código
- **Google Fonts** - Tipografia (Outfit, Caveat)

## 🔧 Como Executar

1. Clone o repositório:
```bash
git clone https://github.com/nevext/algebra-html.git
```

2. Abra o arquivo `index.html` no navegador ou use um servidor local:
```bash
# Com Python
python -m http.server 8000

# Com Node.js (npx)
npx serve
```

3. Acesse `http://localhost:8000` no navegador
